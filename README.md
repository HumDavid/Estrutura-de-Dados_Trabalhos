# TRABALHO DE ESTRUTURA DE DADOS
## Trabalho 1
Implementação e aplicação prática de pilhas, filas, deques e listas encadeadas para resolver problemas clássicos de manipulação de dados e algoritmos básicos.

## Trabalho 2
Implementação de árvores binárias (encadeadas e em array) e desenvolvimento de algoritmos de travessia, verificação estrutural e operações recursivas sobre árvores.

## Trabalho 3
Implementação de árvores de busca binária, árvores AVL e árvores rubro-negras, incluindo inserções, remoções, balanceamento e análise estrutural das árvores resultantes.
